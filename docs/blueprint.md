# **App Name**: Neutral Chat

## Core Features:

- Page Routing: Implement landing, login, signup, chatbot and admin dashboard pages with navigation and simulated route protection.
- Admin Authentication: Create login form for admin, authenticating against hardcoded credentials, and redirect to dashboard upon success.
- Chat Interface: Build chat interface with user and AI message bubbles, scrollable history, and input/send components.
- Simulated Chat Responses: Employ a tool: Upon message submission, a simulated AI response should intelligently select from canned responses.
- Content Upload: Design an admin dashboard for uploading voice (MP3/WAV) and DOCX knowledge files.
- Responsive Layout: Design fully responsive layouts across all pages for mobile and desktop devices.

## Style Guidelines:

- Primary color: A vibrant, energetic, and modern Red (#FF0000).
- Background color: Very light and desaturated Red (#F2E6E6) creating a clean backdrop.
- Accent color: Contrasting vivid Orange (#FFA500) for interactive elements, call to action and highlights.
- Body and headline font: 'Inter' sans-serif for a clean, modern aesthetic. This font works well for both headings and body text.
- Use minimalist icons throughout the application for a clean and modern user experience.
- Utilize a clean and minimal layout with ample whitespace for enhanced readability and visual clarity.
- Implement subtle transitions and animations to enhance user engagement and provide visual feedback.
